日志:
  1. 将 Switch 置于 RCM 中，并将其连接到您的设备
  2. 选择示例有效负载，或上传一个
  3. 点击 '发送 Payload!'
  4. 在出现的同意屏幕上，选择“APX”并点击确认
  5. 如果一切顺利，Payload将成功注入


Mod of WebFG 是从 fusee-launcher. 移植过来的，感谢 atlas44 提供的 java 端口、rajkosto 提供的许多有效负载、ktemkin 和 ReSwitched 提供的 Fusée Gelée 以及大量其他东西！
  • 不保证所有设备能用
  • T只能在chrome浏览器上使用
  • This does NOT currently work on any browser but Chrome, because they don't implement WebUSB.
  • 在 Linux 上，您可能会收到访问被拒绝的错误！如果这样做，您可以尝试在以下位置创建文件 /etc/udev/rules.d/50-switch.rules
    包含以下内容：
    SUBSYSTEM=="usb", ATTR{idVendor}=="0955", MODE="0664", GROUP="plugdev"
